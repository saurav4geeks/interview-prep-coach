# LeetCode Problem Bank

Assign real problems from this bank by topic. Order within each topic is roughly easy → hard. Pick the next unsolved one for the day; save Hard-tagged ones for weekends or when the learner is ready. Always give the problem by name + number + link so the user can open it directly: `https://leetcode.com/problems/<slug>/`.

When picking: match difficulty to the learner's current confidence on that topic (from the tracker). If they're rusty, start one notch easier. On test weekends, pick problems they have NOT seen.
